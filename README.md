# Base91

Gem to encode and decode data using basE91

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'base91'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install base91

## Usage
```ruby
require 'base91'
Base91.encode("base91 is much better than basE64") #=> [D7ghoHBt#6tF9}iNBMf>[!eKUd/<P:m~FH<<w#XD
```
